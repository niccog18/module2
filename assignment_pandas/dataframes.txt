What Is a DataFrame?

A DataFrame is pandas’ core data structure. Think of it as a spreadsheet in Python.
import pandas as pd

data = {
    "name": ["Alice", "Bob", "Charlie", "Diana"],
    "department": ["Engineering", "Marketing", "Engineering", "Sales"],
    "salary": [95000, 72000, 110000, 68000],
    "years": [5, 2, 8, 1],
}

df = pd.DataFrame(data)
print(df)

Output:
      name   department  salary  years
0    Alice  Engineering   95000      5
1      Bob    Marketing   72000      2
2  Charlie  Engineering  110000      8
3    Diana        Sales   68000      1

Filtering: the Power Move 
high_earners = df[df["salary"] > 80000]

engineers = df[df["department"] == "Engineering"]

# Combined conditions: use & (not 'and'), wrap each in parentheses
senior_engineers = df[(df["department"] == "Engineering") & (df["salary"] >= 90000)]


Sorting

Grouping and Aggregation
-This is where pandas becomes essential:
dept_avg = df.groupby("department")["salary"].mean()

dept_summary = df.groupby("department").agg(
    avg_salary=("salary", "mean"),
    max_salary=("salary", "max"),
    headcount=("name", "count"),
    avg_years=("years", "mean"),
)
* This kind of analysis would take 20+ lines of loop-based Python, done in one clear expression.
